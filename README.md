# :zap: Image Editor :zap:
This is a image viewer, made with Tkinter and PIL in python.

## How to execute it?
1. Just install [Python 3.11.3](https://www.python.org/downloads/)
2. Download files
3. Execute main.py

## How to use it?
1. First click on the button that says "Folder".
